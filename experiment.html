Enzyme Device

Whole device

Our project is completed if our device releases different output depending on the order coded on Input. We confirmed that by electrophoresis with input that codes "A then B" and "B then A". DNA strands and enzymes were carefully designed and chosen.(For the details check Method and Material page.) TAMRA (a fluorescence molecule) was modified with Output A while nothing was modified with Output B. Only 2 bands, Output A and Transducer A, could be detected before staining gel. After stainig with SYBR Gold, bands which indicate Output B could be also observed. All enzymes and sets of DNA (Transducer, Gate) were mixed at 45 ℃ , then the reaction starts by adding Input. Reaction time for operating the device was set variously (1, 2, 5, 10, 12, 15, 20, 25, 30, 60, 100 min) to evaluete time variation of Output. After that, the temperature was heated up to 80 ℃ to stop enzymes working then cooled down to the room temperature.
The result with Input AB is following fig.1. That had not stained yet.


fig.1 Time variation of Output A with Input AB (before staining)

The band intnsity of Output A increases between 1~2 min while that of Transducer A decreases.
In the next result, all condition is same to fig.1 except for the Input sequence. That is changed to Input BA.


fig.2 Time variation of Output A with Input BA (before staining)

The band of Output A can be observed from 12~15 min. Therefore, the Output A releasing time is later than the case of Input AB.
We did quantitative analysis about these band intensity. In addition, following graph shows the raito of Output A to the total fluorescence (intensity Transducer A + Output A) with various reaction times. That raito is based on the band intensity shown in fig.1 and fig.2.


fig.3 The raito of Output A with various reaction times

Output A sturated at 2 min when Input codes "A then B" whike that time is 20 min in the case of Input BA. Thus, time interval of them is about 18 minutes. The raito of Output A is saturated aroud 70 % because Output A with TAMARA would form Transducer A again when the solution cooled down from 80 ℃ to room tempereature.
In the discussion page, we are going to discuss about the differnce between this experimental result and simulation result.
Follwing 2 images shows stained gels of fig.1 and fig.2.


fig.4 Time variation of Output A and Output B with Input AB(after staining)

fig.5 Time variation of Output A and Output B with Input BA(after staining)

The band intensity of Output coded as second domain increased gradually after the increassing of first domain. This difference is coursed by the sequences of Input only. Therefore, we confirmed the whole function of our device that releases Output depending on the codes written in Input by this experiment.

Method and Materials

Components (3 processes)

 All partial processes work properly as shown in this section. Reaction temperature (45 ℃) were decided based on following experiments.

Reading Process

This process should work if Input, Template, Polymerase, Nickase, and dNTP are all in the same solution. If this process works properly, the signal will be amplified.

The result of electrophoresis is shown below.


fig.6 Confirmation of Reading process
Other components include Input/Template complex, Polymerase, Nickase, and dNTP.

The band of Signal strand apperared on lane2 while that did not appear on lane3. Unlike lane1, Signal strand was not contained in the sample of lane3. The band of Signal strand appeared by the function of Polymerase and Nickase. Reading process works precisely when the reaction temperature is 37 ℃ ~45 ℃ .

Method and Materials

Releasing Process

In this process, existing Output should be released by both the hybridization of Signal strand and the displacement of polymerase.

The results of electrophoresis is shown below.


fig.7 Confirmation of Releasing process
Other components include Output transducer, Polymerase, and dNTP.

Output did not appear on lane1 in which Signal was not contained. Output appeared on lane2 and lane3 in which Siganl was contained. From this result, we can conclude that Releasing process works correctly. Lane2 and lane3 differ on the reaction temperature, the band of Output appeared on both lane2 and lane3. Releasing process would work precisely when the reaction temperature is between 37 ℃ and 45 ℃ .

Method and Materials

Updating Process

In this process, Updater is released by both the hybridization of Signal and the displacement of polymerase. Recognition sequence is formed by the hybridization of Updater and Input, then EcoRI cuts the Input.

The result of electrophoresis is shown below.


fig.8 Confirmation of Updating process
Other components include Input, Gate, Polymerase, EcoRI, and dNTP.

From the result of lane1 and lane2, we found that Updated complex and Waste are generated when Signal is entered. Therefore, Updating process works correctly by the function of EcoRI and Polymerase.

Combination of 3 processes

3 processes with Polymerase, Nickase, and restriction enzyme need to work in the same H buffer as we designaed. The release of first output was confirmed in this overall device. In addition, how long did it take to release first Output was evaluated by this experiment.
Therefore, enzymatic reaction times were set variously.
The release of the first output correspond to Input AB, that is Output A, was confirmed in this experiment. Output A is modified with fluorescent molecule TAMRA. Before stained by SYBR Gold, a band contains Output A (Transducer, Output) must be able to be observed.
The result of electrophoresis is shown below.


fig.9 Result of electrophoresis to confirm the first output releasing (before staining)

The time variation of fluorescence intensity of Output band is shown in the graph below.


fig.10 The time variation of fluorescence intensity of Output A

Fluorescent intensity of Output was increased as reaction time became longer. The result of lane8 can be regarded as an outcome when reaction time is 0 minutes. The graph above shows the ratio of fluorescence intensity of Output in lane 1 to 7, when the fluorescence intensity sum of Output and Transducer in each lanes set to 100%. Although about 10% of Output exists at the initial stage of reactions, it can be thought to be the Output that missed to form Transducer when annealing. Concentration of this Output must be able to decrease by refining the solution. Output was saturated about 10 minutes later. In the end, it became clear that 50 to 60% of fluorescent molecule exists from the first is released.

Figure below shows the result of electrophoresis above. DNA is stained with SYBR Gold.


fig.11 Result of electrophoresis to confirm the first output releasing (after staining)

When reaction time is over 20 minutes, Signal band emerged. The reason can be thought that Output discharged in Releasing process saturated, and Signal exceeded. This excess Signal is used in next Updating process.

Enzyme free device

Whole device

It is necessary to confirm that enzyme free device we designed really releases Output depends on Input sequences in order. This experiment measures the intensity of FAM (modified with Output A). We prepared Input AB and Input BA. The order of releasing would be changed depending on these Inputs if our device works properly.
Following graph shows the result.


fig.12 The intensity of Output A (FAM). The fluorescence wavelength is 520nm. To do quantitative analysis the highest intensity data was defined 100% while the lowest was defined 0 %.

According to the left graph, the intensity of Output A coded as second domain was increased later than that was first domain. Therefore, this result shows our device releases Output based on the code written in Input. The time difference between first and second releasing is about 40 minutes.
We are going to discuss about this result along with simulation.

Component (Seesaw gate)

We checked the function of one seesaw gate by measuring fluorescence intensity and doing electrophoresis.
If first Output is released, we can conclude that one Seesaw gate works properly.

Measurement of fluorescence

Output is modified with FAM. The DNA that hybridizes with Output previously is modified with BHQ. Click here for more information about FRET. Input is designed to release A then B. FAM is modified with Output A. The final concentration of all components (Input, Transducer, Gate, Trigger) are 100nM.
Time change of fluorescence is shown in following figure. 0[s] is the time when Trigger was added into the solution in which Input, Transducer, and Gate were mixed.


fig.13 Time change of Output A measured by spectrofluorometer

To do quantitative analysis, 100% was defined the fluorescence intensity level of DNA modified FAM, and 0% was the level of the solution with Input, Transducer, and Gate. After adding Trigger, strength of fluorescence increased gradually. Therefore, Output A was released by the function of the device properly. The increase of fluorescence stopped at about 1700sec. Probably the reaction ended at this point. Half of Output strands were released according to the fluorescence intensity.
